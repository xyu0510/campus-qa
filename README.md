# 厦门工学院 - 校园办事问答系统

🎓 厦门工学院专属校园办事问答助手，为学生提供便捷的校园业务咨询服务。

## 功能特点

- 🤖 智能问答：解答证件补办、报销、场地申请等校内业务问题
- ⭐ 收藏功能：收藏常用问答，方便随时查阅
- 📂 分类导航：按业务类型快速查找
- 💬 反馈机制：支持问答反馈，持续优化

## 页面说明

| 页面 | 说明 |
|------|------|
| `index.html` | 主页入口 |
| `campus-qa.html` | 校园问答主界面 |
| `login.html` | 登录页面 |
| `admin.html` | 管理后台 |

## 本地运行

```bash
# 使用 Python 启动本地服务器
python -m http.server 8000

# 访问地址
http://localhost:8000
```

## 部署到 GitHub Pages

1. 创建 GitHub 仓库
2. 上传所有文件
3. 在仓库 Settings → Pages 中启用 GitHub Pages
4. 选择 main 分支，保存
5. 访问地址：`https://你的用户名.github.io/仓库名/`

## 技术栈

- HTML5 + CSS3 + JavaScript
- 微信小程序风格 UI
- 响应式设计，支持移动端
