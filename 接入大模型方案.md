# 问答界面接入大模型方案

## 🎯 方案概述

将校园办事问答系统接入大模型，实现智能问答功能。

---

## 📋 方案选择

### 方案一：前端直接调用（简单快速）⭐ 推荐测试

**优点：**
- ✅ 无需后端服务器
- ✅ 部署简单
- ✅ 适合快速测试

**缺点：**
- ⚠️ API密钥暴露风险
- ⚠️ 有跨域限制

**适用场景：** 个人测试、内部使用

---

### 方案二：后端代理调用（安全稳定）⭐ 推荐生产

**优点：**
- ✅ API密钥安全
- ✅ 无跨域问题
- ✅ 可添加鉴权
- ✅ 可记录日志

**缺点：**
- ⚠️ 需要后端服务器
- ⚠️ 部署相对复杂

**适用场景：** 正式上线、对外服务

async function searchQuestion() {
    const question = document.getElementById('questionInput').value.trim();
    
    if (!question) {
        alert('请输入您的问题');
        return;
    }

    // 显示加载状态
    const answerContent = document.getElementById('answerContent');
    const placeholder = document.getElementById('placeholder');
    
    placeholder.style.display = 'none';
    answerContent.classList.add('show');
    answerContent.innerHTML = '<div style="text-align: center; padding: 40px;"><p style="font-size: 18px; color: #667eea;">🤖 AI正在思考中...</p></div>';

    try {
        // 调用大模型API
        const response = await fetch(AI_CONFIG.apiUrl, {
            method: 'POST',
            headers: {
                'Content-Type': 'application/json',
                'Authorization': `Bearer ${AI_CONFIG.apiKey}`,
            },
            body: JSON.stringify({
                model: AI_CONFIG.model,
                messages: [
                    {
                        role: 'system',
                        content: KNOWLEDGE_CONTEXT
                    },
                    {
                        role: 'user',
                        content: question
                    }
                ],
                temperature: 0.7,
                max_tokens: 1000
            })
        });

        const data = await response.json();
        
        // 解析返回结果（根据不同大模型调整）
        let answer = '';
        if (data.output) {
            // 通义千问格式
            answer = data.output.text || data.output.choices[0].message.content;
        } else if (data.choices) {
            // OpenAI/DeepSeek格式
            answer = data.choices[0].message.content;
        } else if (data.result) {
            // 文心一言格式
            answer = data.result;
        }

        if (answer) {
            // 格式化显示答案
            answerContent.innerHTML = `
                <h3>📋 智能回答</h3>
                <div style="font-size: 15px; color: #555; line-height: 1.8; white-space: pre-wrap;">
                    ${answer}
                </div>
                <div style="margin-top: 20px; padding-top: 15px; border-top: 1px dashed #e0e0e0; font-size: 12px; color: #999;">
                    💡 此回答由AI生成，仅供参考，具体以学校官方规定为准
                </div>
            `;
        } else {
            throw new Error('未获取到有效回答');
        }
        
    } catch (error) {
        console.error('查询失败:', error);
        answerContent.innerHTML = `
            <div style="text-align: center; padding: 40px;">
                <p style="font-size: 40px; margin-bottom: 20px;">⚠️</p>
                <p style="font-size: 16px; color: #999;">查询出错：${error.message}</p>
                <p style="font-size: 14px; color: #aaa; margin-top: 10px;">请检查网络连接或API配置</p>
            </div>
        `;
    }
}

## 🔧 方案二：后端代理调用

### 步骤1：创建后端服务

#### 使用Node.js + Express：

```javascript
// server.js
const express = require('express');
const cors = require('cors');
const fetch = require('node-fetch');

const app = express();
app.use(cors());
app.use(express.json());

// 大模型配置
const AI_CONFIG = {
    apiUrl: 'https://dashscope.aliyuncs.com/api/v1/services/aigc/text-generation/generation',
    apiKey: '你的API密钥', // 安全存储在后端
    model: 'qwen-turbo',
};

// 知识库上下文
const KNOWLEDGE_CONTEXT = `...`; // 同前端版本

// 问答接口
app.post('/api/chat', async (req, res) => {
    try {
        const { question } = req.body;
        
        const response = await fetch(AI_CONFIG.apiUrl, {
            method: 'POST',
            headers: {
                'Content-Type': 'application/json',
                'Authorization': `Bearer ${AI_CONFIG.apiKey}`,
            },
            body: JSON.stringify({
                model: AI_CONFIG.model,
                messages: [
                    { role: 'system', content: KNOWLEDGE_CONTEXT },
                    { role: 'user', content: question }
                ]
            })
        });

        const data = await response.json();
        res.json(data);
        
    } catch (error) {
        res.status(500).json({ error: error.message });
    }
});

app.listen(3000, () => {
    console.log('服务器运行在 http://localhost:3000');
});
```

## 💰 成本估算

### 免费额度对比：

| 大模型 | 免费额度 | 预计问答量 |
|--------|---------|-----------|
| 通义千问 | 100万tokens/月 | ~5000次问答 |
| 文心一言 | 较大 | ~3000次问答 |
| 智谱AI | 较大 | ~3000次问答 |

### 成本优化建议：

1. **缓存常见问题** - 减少API调用
2. **限制回答长度** - 设置max_tokens
3. **使用更便宜模型** - 如qwen-turbo
4. **批量处理** - 减少请求次数
